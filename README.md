## Setup

We're using Ubuntu. On a fresh installation, login as root and run

  ```sh
  ./setup.sh
  ```

## Webapp

  ```sh
  node server/server.js
  ```

## Testing

  ```sh
  ionic build ios
  ionic emulate ios
  ```
